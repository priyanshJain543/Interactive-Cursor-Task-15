# Interactive Cursor Project 🚀

An immersive web interface that explores the synergy between **interactive design** and **minimalist aesthetics**. This project features a high-performance custom cursor implementation and a motivational tribute to **Dr. A.P.J. Abdul Kalam**, emphasizing fluid user experience (UX) and modern front-end techniques.

Explore the interactive experience here: **[Live Demo Link]()** 🌐

## 💎 Key Highlights

- **Custom Interaction Layer:** Engineered a bespoke circular cursor using Vanilla JavaScript to replace the default system pointer, enhancing the interface's unique identity.
- **Advanced Blending Modes:** Leveraged `mix-blend-mode: difference` to implement a sophisticated color-inversion effect, ensuring high contrast and readability across various UI elements.
- **Modern Visual Stack:** Utilized a 135° linear gradient and the **Montserrat** typeface to establish a strong visual hierarchy and contemporary look-and-feel.
- **Responsive Architecture:** Implemented fluid typography using Viewport Units (`vw`) to maintain design integrity across diverse display resolutions.

## 🛠️ Technical Stack

- **Architecture:** Semantic HTML5 for structural integrity.
- **Styling:** CSS3 featuring Flexbox layouts and Google Fonts API integration.
- **Logic:** Event-driven JavaScript for real-time DOM updates and coordinate tracking.
